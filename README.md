# Beyond Stochasticism: The OEA Framework
This repository contains the research design, experimental protocol, and publication package for:
"Beyond Stochasticism: An Ontological Framework for Agentic Stability."

## Project Goal
Establish and empirically test the OEA (Ontology, Epistemic, Agentic) framework as a philosophical and engineering guardrail against model collapse in recursive agentic systems.

## Repository Structure
- `docs/research-agent-spec.md`: Canonical OEA research agent specification.
- `docs/manifesto.md`: "Architecture of Meaning" opening manifesto.
- `docs/methodology.md`: OEA tri-layer process and trust-aware orchestration protocol.
- `experiments/specsmith-protocol.md`: Experimental methods aligned to `specsmith` (`develop` branch).
- `arxiv/main.tex`: LaTeX scaffold for arXiv/PhilSci-Archive packaging.
- `arxiv/references.bib`: Citation skeleton for required references.
- `distribution/distribution-strategy.md`: Publication and dissemination plan.

## Core References
- Fu et al., 2025 (model collapse / recursive degradation / divergence framing)
- Roumeliotis et al., 2025 (trust-aware orchestration / epistemic reliability)

## Suggested Workflow
1. Implement experiment harness in `specsmith`-compatible environment.
2. Run recursive stability and epistemic friction experiments.
3. Export result tables and figures to `arxiv/`.
4. Finalize manuscript and submit to PhilSci-Archive + OpenReview.
