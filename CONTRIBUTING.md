# Contributing

## Scope
This repository hosts the OEA framework research specification, experiment protocols, and manuscript assets.

## Contribution Guidelines
1. Open an issue before substantial changes.
2. Keep experimental assumptions explicit and falsifiable.
3. Preserve source-traceability for any claim added to manuscript sections.
4. Use clear commit messages scoped to one conceptual change.

## Pull Request Expectations
- Reference related issue IDs.
- Include rationale for methodology changes.
- Include reproducibility notes when experiments or metrics change.
